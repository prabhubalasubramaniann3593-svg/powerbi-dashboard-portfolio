# Healthcare Analytics Dashboard

This Power BI dashboard provides insights into healthcare operations and performance metrics.

## Key Metrics
- Patient Volume
- Treatment Success Rate
- Operational KPIs
- Regional Healthcare Insights

## Features
- Interactive filters and slicers
- Drill-through analysis
- KPI cards and trend charts
- Data modeling using star schema

## Tools Used
- Power BI
- DAX
- SQL
- Power Query

## Files
- healthcare.pbix → Power BI report file
